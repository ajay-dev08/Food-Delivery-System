# 🍔 FoodExpress - Food Delivery System

FoodExpress is a modern and responsive Food Delivery website developed using HTML, CSS, and JavaScript. Users can browse food items, search meals instantly, add products to cart, and complete orders through an interactive checkout payment popup.

---

# 🚀 Features

- Responsive modern UI design
- Food search functionality
- Add to cart system
- Remove cart items
- Dynamic cart total calculation
- Checkout payment popup
- Delivery address form
- Mobile-friendly layout
- Smooth scrolling effects
- Special offers section

---

# 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

# 📸 Modules

## 🏠 Home Section
Modern hero section with attractive food banner.

## 🍕 Food Menu
Displays available food items with price and cart options.

## 🔍 Search Feature
Users can instantly search foods by name.

## 🛒 Cart System
Add/remove items dynamically with live total updates.

## 💳 Payment Popup
Interactive checkout form with payment details and delivery address.

## 🎁 Offers Section
Shows special discounts and combo offers.
